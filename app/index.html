Deployed on AWS

Infrastructure as Code,
end to end.

A static site provisioned by Terraform, served by Nginx across an Auto Scaling Group, balanced by an Application Load Balancer, and shipped by GitHub Actions.

How it works Get in touch

The stack

Terraform

Six reusable modules — VPC, security groups, IAM, launch template, ALB and Auto Scaling — with remote state in S3 and locking via DynamoDB.

Auto Scaling

Runs a minimum of 1 and a desired 2 instances, scaling out to 4 when CPU passes 70% and back in below 30%.

GitHub Actions

Validates HTML and Terraform, plans the change, builds the artifact, deploys over SSH, then health-checks the load balancer.

Observability

CloudWatch tracks CPU, memory, network and status checks, with alarms wired to the scaling policies.
